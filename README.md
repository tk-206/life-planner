3. 포기하지 마라.
4. 약속한 것은 무조껀 지키자

6. 성공해서 보답하자!
7. 물에 빠져 죽기 직전에 숨을 쉬고 싶다는 생각의 간절함 만큼의 간절함으로 임해야 성공한다.

10. 비교는 즐거움의 도둑이다. 남과 비교하지 말자.
11. 넘어져도 일어서기.

16. 바늘구멍도 구멍은 있다.
17. 모르는 사람도 이해하기 쉽게 설명해주면 좋다.
18. 나는 운이 좋은 사람이다.
19. 배우는 것을 두려워하지말자.
20. 후회할거면 죽기 직전까지 노력해봤는지 생각해보자.
21. 오늘 할 일을 내일로 미루지 말자.
22. 믿어주는 사람에게 보답하자.
